## Usage

> Visit [our site](twitchlite.herokuapp.com)
> Click on an image to load a highlight, or log in with your Twitch account for a more personalized feed.
> You can also use the search bar to narrow down the results.

## Requirements

- Node 6.6.0
- npm 3.10.8
